Explication des attributs

  • src : Image de fallback pour les navigateurs qui ne supportent pas srcset
  • srcset : Liste des images disponibles avec leur largeur (ex: "photo-320w.webp 320w")
  • sizes : Indique au navigateur quelle taille d'image utiliser selon la largeur de l'écran
  • loading="lazy" : Charge l'image uniquement quand elle est proche de la zone visible

Breakpoints recommandés

  • 320w : Petits mobiles
  • 640w : Grands mobiles / petites tablettes
  • 1024w : Tablettes / petits écrans
  • 1920w : Desktop / grands écrans
